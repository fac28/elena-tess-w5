## Installation

Clone the project

```bash
  git clone https://github.com/fac28/elena-tess-w5.git
```

Go to the project directory

```bash
  cd elena-tess-w5
```

Install dependencies

```bash
  npm install
```

## Usage

Start the server

```bash
  npm run dev
```
